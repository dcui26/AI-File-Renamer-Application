frontend:
    allow user to select folder (define a path)
    have a button to start processing images
    optional cancel button
    extra: progress bar
backend:
    read image from selected folder
    send images to api call
    rename files safely
    extra: return progress/status to ui

considering adding a sort.py to sort by name into new folders created

project/
│
├── backend/                 # all logic and file processing
│   ├── __init__.py
│   ├── file_safety.py        # functions to rename files, check extensions
│   ├── pdf_methods.py         # functions to convert PDFs to images
│   ├── ai_reader.py         # functions to send images to AI for reading
│   ├── config.py            # all constants (supported extensions, date format, etc.)
│
├── ui/                      # user interface
│   ├── __init__.py
│   ├── app.py               # GUI with folder picker and start button
│
├── assets/                  # optional, for sample images or PDFs
│
├── poppler/                 # include Poppler binaries here (for PDF conversion)
│   └── bin/                 # pdftoppm.exe, etc. (Windows only)
│
└── run.py                   # main entry point: starts the UI
